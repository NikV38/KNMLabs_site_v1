# KNM Labs Website

A modern, single-page website for KNM Labs - a software engineering studio focused on building reliable, maintainable systems.

## Features

- **Console-style interface** with dark theme and glassmorphism effects
- **Smooth scroll animations** with staggered reveals and parallax effects
- **Interactive elements** with 3D tilt effects on hover
- **Responsive design** optimized for all screen sizes
- **Background image integration** with layered gradients and blur effects
- **Contact form** for project inquiries

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, gradients, backdrop filters, animations
- **Vanilla JavaScript** - Intersection Observer API, smooth scrolling, interactive effects
- **IBM Plex Mono** - Google Fonts typography

## File Structure

```
website_v1/
├── index.html      # Main HTML structure
├── styles.css      # All styling and animations
├── scripts.js      # Interactive functionality
├── background.jpg  # Background image
└── README.md       # This file
```

## Setup

1. Clone or download the repository
2. Ensure `background.jpg` is in the root directory
3. Open `index.html` in a modern web browser
4. No build process or dependencies required

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Requires support for:
  - CSS Grid & Flexbox
  - CSS Custom Properties
  - Backdrop Filter
  - Intersection Observer API

## Customization

- **Colors**: Edit CSS custom properties in `styles.css` (`:root` section)
- **Content**: Update sections in `index.html`
- **Animations**: Modify timing and effects in `styles.css` and `scripts.js`
- **Background**: Replace `background.jpg` with your own image
